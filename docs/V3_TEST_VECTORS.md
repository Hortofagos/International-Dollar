# IND V3 Test Vectors

The authoritative executable vectors live in `tests/test_crypto_ed25519.py`,
`tests/test_keys_v3.py`, `tests/test_binary_v3.py`, `tests/test_protocol_v3.py`,
`tests/test_archive_segment_v3.py`, `tests/test_proof_bundle_v3.py`, and
`tests/test_spend_map_v3.py`.

The object vectors are generated by the field-level V3 binary encoders. The old
canonical JSON-in-envelope form is intentionally rejected as malformed V3
binary, so no compatibility vector is kept for it.
